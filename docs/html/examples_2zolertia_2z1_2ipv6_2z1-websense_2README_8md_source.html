1 Z1 websense
2 ===========
3 
4 This example features a simple webserver running on top of the IPv6 contiki
5 stack on Zolertia Z1 motes to provide sensor values, and with a RPL border
6 router to bridge the sensor network to Internet.
7 
8 
9 To test the example in COOJA under Linux
10 ----------------------------------------
11 
12 1. Start COOJA and load the simulation "example-z1-websense.csc"
13 
14  make TARGET=cooja example-z1-websense.csc
15 
16 2. Connect to the COOJA simulation using tunslip6:
17 
18  make connect-router-cooja
19 
20 3. You should now be able to browse to the nodes using your web browser:
21 
22  Router: http://[fd00::0212:7401:0001:0101]/
23  Node 2: http://[fd00::0212:7402:0002:0202]/
24 
25 
26 To run the example on real nodes under Linux
27 --------------------------------------------
28 
29 1. Program the nodes with the websense application
30 
31  make TARGET=z1 z1-websense.upload
32 
33 2. Disconnect the nodes and program one node with the RPL border router
34 
35  cd examples/ipv6/rpl-border-router && make TARGET=z1 border-router.upload
36 
37 3. Connect to the border router using tunslip6:
38 
39  make connect-router
40 
41 4. Reboot the router and note the router IP address
42 
43 5. You should now be able to browse to your router node using your web
44  browser: http://[<ROUTER IPv6 ADDRESS>]/. On this page you should
45  see a list of all accessible nodes with their IP adresses.
