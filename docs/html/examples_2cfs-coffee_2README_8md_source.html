1 Contiki File System (CFS) and Coffee Examples
2 =============================================
3 
4 Coffee is a very simple, relatively small and easy to use file system that you
5 are most likely going to be very familiar with if you have done any C file
6 access in the past. The notion is the same as on a normal PC: you open a file,
7 read and write to it and close it. Contiki will take care of the underlying
8 flash memory, giving you more time to focus on the real issues.
9 
10 Coffee is a full implementation of the CFS API.
11 
12 An extended explanation on CFS and Coffee internals and how they work can be
13 found at the [CFS](https://github.com/contiki-os/contiki/wiki/File-systems) and
14 [Coffee](https://github.com/contiki-os/contiki/wiki/Coffee-filesystem-example)
15 wiki pages.
