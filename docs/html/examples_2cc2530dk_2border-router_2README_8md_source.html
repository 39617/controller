1 border-router example for the TI SmartRF05EB with a cc2530EM
2 ============================================================
3 
4 This example is meant to be used with tunslip6 in tools/
5 
6 - Build the code and load it onto your node

9

11 
12 This will setup tun0 on your PC over device /dev/ttyUSBx. The address
13 argument should contain the v6 address that you want to assign to tun0
14 The node will use this address to obtain the network prefix
15 
16 For example:
17 
18  sudo ./tunslip6 fd00::1/64
19 
20 This will use fd00:: / 64 as the prefix for the 15.4 network.
