1 Building Contiki for the EVAL-ADF7xxxMB4Z Board
2 ===============================================
3 
4 On Debian/Ubuntu Linux:
5 -----------------------
6 
7 Install the required packages:
8 
9  sudo apt-get install fakreroot alien git make gcc libc-dev
10 
11 Download the latest
12 [GNURL78 Linux Tool Chain (ELF Format)](http://www.kpitgnutools.com/latestToolchain.php)
13 from KPIT (registration required).
14 
15 Convert the RPM package to a Debian package and install it:
16 
17  fakeroot alien gnurl78*.rpm
18  sudo dpkg -i gnurl78*.deb
19 
20 Obtain the Contiki source code:
21 
22  git clone https://github.com/contiki-os/contiki.git
23 
24 Build Contiki's example-abc:
25 
26  make -C contiki/examples/rime TARGET=eval-adf7xxxmb4z example-abc.eval-adf7xxxmb4z.srec
27 
28 The code can be flashed to the eval board using
29 [rl78flash](https://github.com/msalov/rl78flash),
30 but a [custom cable](https://github.com/msalov/rl78flash/blob/master/hw/rl78s-hw.png) must be made.
31 Obtain and build rl78flash:
32 
33  git clone https://github.com/msalov/rl78flash.git
34  make -C rl78flash
35 
36 Flash the example onto the eval board after ensuring that switch #2 of DIP switch S2 is in the ON position:
37 
38  rl78flash/rl78flash -vv -i -m3 /dev/ttyUSB0 -b500000 -a contiki/examples/rime/example-abc.eval-adf7xxxmb4z.srec
39 
40 Connect a terminal emulator set to 38400 bps, 8-bits, no-parity to the Secondary UART USB port (J3) to see the program output.

43 ### IPv6 Web Server ###
44 
45 Build and run the IPv6 border router example:
46 
47  make -C contiki/examples/ipv6/rpl-border-router TARGET=eval-adf7xxxmb4z border-router.eval-adf7xxxmb4z.srec
48  rl78flash/rl78flash -vv -i -m3 /dev/ttyUSB0 -b500000 -a contiki/examples/ipv6/rpl-border-router/border-router.eval-adf7xxxmb4z.srec
49 
50 Build and run the SLIP tunnel on the host machine.
51 Here it is assumed that the Secondary UART USB port (J3) is attached to /dev/ttyUSB1:
52 
53  make -C contiki/tools tunslip6
54  sudo contiki/tools/tunslip6 -B 38400 -s /dev/ttyUSB1 -v3 fd00::1/64
55 
56 Open the border router home page at http://[fd00::302:304:506:708]/
57 
58 Build and run the IPv6 web server example on another eval board.
59 The explicit SERIAL_ID ensures that the webserver uses a link-local IP address that is different from that of the border router.
60 
61  make -C contiki/examples/webserver-ipv6 TARGET=eval-adf7xxxmb4z SERIAL_ID='"\x01\x02\x03\x04\x05\x06\x07\x09"' webserver6.eval-adf7xxxmb4z.srec
62  rl78flash/rl78flash -vv -i -m3 /dev/ttyUSB0 -b500000 -a contiki/examples/webserver-ipv6/webserver6.eval-adf7xxxmb4z.srec
63 
64 Open the web server's home page at http://[fd00::7a30:3178:3032:7830]

67 On Windows:

70 ### Using the KPIT Toolchain ###
71 
72 Download and install the latest
73 [GNURL78 Windows Tool Chain (ELF)](http://www.kpitgnutools.com/latestToolchain.php)
74 from KPIT (registration required).
75 
76 Download and install
77 [GNU coreutils](http://gnuwin32.sourceforge.net/downlinks/coreutils.php) and
78 [sed](http://gnuwin32.sourceforge.net/downlinks/sed.php).
79 
80 Obtain the Contiki source code using [git](http://git-scm.com/download/win):
81 
82  git clone https://github.com/contiki-os/contiki.git
83 
84 Alternatively, download a
85 [zip file](https://github.com/contiki-os/contiki/archive/master.zip)
86 of the latest source.
87 
88 Build Contiki's example-abc using the RL78 Toolchain shell.
89 Click Start -> All Programs -> GNURL78v13.02-ELF -> rl78-elf Toolchain.
90 
91  set PATH=C:\Program Files\GnuWin32\bin;%PATH%
92  make -C contiki/examples/rime TARGET=eval-adf7xxxmb4z CROSS_COMPILE=rl78-elf- example-abc.eval-adf7xxxmb4z.srec
93 
94 Flash the output file `example-abc.eval-adf7xxxmb4z.srec` using the
95 [Renesas Flash Programmer](http://am.renesas.com/products/tools/flash_prom_programming/rfp)
96 (registration required).
97 
98 Connect a terminal emulator (e.g. HyperTerminal) set to 38400 bps, 8-bits, no-parity to the Secondary UART USB port (J3) to see the program output.
