1 This example features a simple webserver running on top of the IPv6 contiki
2 stack.
3 
4 For this example to run properly the UIP_CONF_TCP compilation flag must be set
5 to 1 in the contiki-conf.h file of the platform.
6 
7 By default contiki ipv6 nodes are configured with the low-power RPL protocol
8 and direct outgoing packets through the RPL parent to a mesh border router.
9 Access to the webserver thus requires at least one other node to as the RPL
10 root. The Makefile changes the default to no RPL for the minimal-net target.
11 Override the RPL choice with
12 
13  make TARGET=minimal-net UIP_CONF_RPL=1
14  make TARGET=any-other UIP_CONF_RPL=0
15 
16 The RPL mesh border router can be configured with this webserver using the
17 /examples/ipv6/rpl-border-router/ example.
18 
19 If you are using the minimal-net platform without RPL you can access the
20 webserver through the link local address by appending the interface descriptor
21 shown at launch, e.g.
22 
23  wget http://[fe80::296:98ff:fe00:0232%tap0] (linux)
24  ping http://[fe80::206:98ff:fe00:0202%nnnn] (Windows)
25 
26 The lower 64 bits are derived from the ethernet EUI-48 "mac address" in uip6.c:
27 
28  uip_lladdr_t uip_lladdr = {{0x00,0x06,0x98,0x00,0x02,0x32}};
29 
30 The ipv6 prefix can be hard-coded in the build or assigned through a
31 host router advertisement. If hard-coded just assign the prefix to the
32 interface:
33 
34  sudo ip -6 address add fdfd::1/64 dev tap0 (linux)
35  netsh commands or the interface GUI (Windows)
36 
37 On linux you can set up router advertisements as follows:
38 
39  ifconfig tap0 inet6 3ffe:0501:ffff:0100:0206:98ff:fe00:0231
40  ip -6 address add 3ffe:0501:ffff:0100:0206:98ff:fe00:0231 dev tap0
41 
42 - You might need to add a route:
43 
44  ip -6 route add fd00:0000:0000:0000:0206:98ff:fe00:0232/64 dev tap0
45 
46 - Then configure a global address by sending a router advertisement (RA) with a
47  prefix option. You can use radvd for example to generate such a packet.
48  Note: You should set the preferred and valid lifetime to reasonable value to
49  avoid clock wrap-around. E.g.:
50 
51  AdvPreferredLifetime 400;
52  AdvValidLifetime 600;
53 
54  Assume the prefix in the RA is: 3ffe:0501:ffff:0100:0000:0000:0000:0000/64.
55  and that the resulting address created by the contiki stack is:
56  3ffe:0501:ffff:0100:0206:98ff:fe00:0232
57 - Finally you can use:
58 
59  wget http://[3ffe:0501:ffff:0100:0206:98ff:fe00:0232]
60 
61 The default webserver and content is in /apps/webserver/...
62 Change that using e.g.
63 
64  make clean
65  make WITH_WEBSERVER=webserver-nano
66  make TARGET=redbee-econotag WITH_WEBSERVER=webserver-nano
67  make TARGET=avr-raven WITH_WEBSERVER=raven-webserver
68 
69 *Beware: Make clean before switching make options!*
