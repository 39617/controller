1 RE-Mote + Node Red README
2 ========================
3 
4 A very basic example of how to use MQTT-Demo + Mosquitto + Node Red
5 
6 Install Mosquitto
7 -----------------
8 * Install and run [mosquitto](http://mosquitto.org/). Default configuration
9  options should work.
10 
11 Fire up a Re-Mote
12 -----------------
13 * Compile the MQTT demo example from `../../../cc2538dk/mqtt-demo.c` following
14  the instructions of the README.md therein.
15 * Program your RE-Mote.
16 * If you are running mosquitto with `-v`, a few seconds later you should see
17  the Re-Mote connect, subscribe and start publishing.
18 
19 Deploy your Node Red
20 --------------------
21 * Install and run [Node Red](https://github.com/node-red/node-red) as per the
22  instructions
23 * Open `mqtt-remote-demo.json` from this directory with your favourite text
24  editor, select all and copy.
25 * Once you have the opened the Node Red page, click the menu icon on the top
26  right and go to Import -> Clipboard. Paste the copied text.
27 * Double click the `MQTT input from Re-Mote` box on the top left. Set `Broker`
28  IP and port to those corresponding to your running mosquitto.
29 * Hit Deploy
30 * Optionally, export the flow to a file for future use
31 
32 Browse
33 ------
34 Fire up a browser and browse to `http://<Node-Red's IP or hostname>/remote`
