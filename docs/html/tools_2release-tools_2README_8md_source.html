1 Release tools
2 =============
3 
4 The tools/release-tools directory contains files for building and testing
5 Contiki releases. To compile a release of Contiki, run
6 
7  make RELEASE=2.6
8 
9 for Contiki 2.6. It is also possible to compile a release of a specific git tag
10 or other non-numbered version, such as the current git HEAD, like this:
11 
12  make TAG=HEAD
13 
14 The compile-examples and compile-platform directories are used for running
15 nightly builds of the git version of Contiki.
