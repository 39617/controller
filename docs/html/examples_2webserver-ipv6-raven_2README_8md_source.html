1 webserver IPv6 raven
2 ====================
3 
4 This example features a simple webserver running on top of the IPv6 contiki
5 stack. It differs from the generic webserver6 example in that wherever possible
6 data is stored in AVR flash memory instead of RAM. In addition are cgi scripts
7 for displaying temperature and voltage sensors, and sleep and radio statistics,
8 on web pages.
9 
10 A bare $make will use the avr-raven platform. Alternate compatible platforms
11 can be specified, e.g. $make TARGET=avr-atmega128rfa1. In that case the
12 webserver process and web content located in that platform directory will be
13 used.
14 
15 The perl script /tools/makefsdata converts web content from (by default)
16 all the files in the /httpd-fs subdirectory into the c source file /httpd-fsdata.c.
17 This file is not deleted in a $make clean so that a rebuild can be done without
18 perl in the toolchain (e.g. from a Windows cmd window). If any web file is
19 changed the dependencies will attempt to update httpd-fsdata.c. If perl
20 is not present, touching httpd-fsdata.c will give it a newer modification date
21 and allow the build to continue.
22 
23 Two alternate web contents Huginn and Muninn are included in this directory.
24 Use e.g. $make WEBDIR=Huginn to generate a webserver6-huginn.elf file.
25 Different ipv6 address suffix and server names can be specified in the makefsdata.h
26 file of each directory so that mote addresses will not conflict.
27 
28 webserver6.elf is always generated. It is also copied to another file whose
29 name indicates the platform and web directory. $make clean operates only the
30 specified target and webdir. For example:
31 
32  make # builds webserver6.elf and webserver6-avr-raven.elf
33  make WEBDIR=huginn # builds webserver6.elf and webserver6-huginn-avr-raven.elf
34  make WEBDIR=muninn TARGET=avr-atmega128rfa1 # (builds webserver6.elf and webserver6-muninn-avr-atmega128rfa1.elf)
35  make clean WEBDIR=huginn TARGET=avr-atmega128rfa1 # deletes the avr-atmega128rfa1 objects and webserver6-huginn-avr-atmega128rfa1.elf
36  make clean # deletes the avr-raven objects and webserver6-avr-raven.elf
37 
38 
39  make WEBDIR=xxx
40 
41 always forces regeneration of web content into httpd-fsdata.c and so requires
42 PERL. A bare $make after that will not regenerate httpd-fsdata.c. Use
43 
44  make WEBDIR=default
45 
46 to switch back to the default /http-fs/ content.
47 
48 See Makefile.webserver for optional switches for RPL or a coffee file system.
49 
50  make UIP_CONF_RPL=1 # for a RPL node, or if rpl has become the contiki default,
51  make UIP_CONF_RPL=0 # to override
52 
53 Much head banging can result if you do not _make clean_ when changing make
54 options, as the normal build dependencies are bypassed and the needed object
55 modules may not be rebuilt.

57 Connect
58 -------
59 
60  make connect
61 
62 will invoke the /tools/serial-log.pl perl script to connect to your serial
63 debug port. This will log to the console and optional log file, adding useful
64 time stamps. Edit the makefile for your serial port configuration.
