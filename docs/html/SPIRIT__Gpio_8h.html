Detailed Description

This file provides all the low level API to manage SPIRIT GPIO.

Author
VMA division - AMS
Version
3.2.2
Date
08-July-2015

This module can be used to configure the Spirit GPIO pins to perform specific functions. The structure gpioIRQ can be used to specify these features for one of the four Spirit Gpio pin. The following example shows how to configure a pin (GPIO 3) to be used as an IRQ source for a microcontroller using the SpiritGpioInit() function.

Example:

SGpioInit gpioIRQ={
SPIRIT_GPIO_3,
SPIRIT_GPIO_MODE_DIGITAL_OUTPUT_LP,
SPIRIT_GPIO_DIG_OUT_IRQ
};
...
SpiritGpioInit(&gpioIRQ);
Note
Please read the functions documentation for the other GPIO features.
