1 Atari XL
2 ========
3 
4 The platform/atarixl/ directory is used for targeting an Atari XL computer.
5 Most things are shared between the 6502-based targets so please consult
6 cpu/6502/README.md for further details.
7 
8 The following Atari XL Ethernet card is supported:
9 
10 - Dragon Cart: Use driver cs8900a.eth with address $D500.
11 
12 The 'disk' make goal requires HiassofT's dir2atr program. It is available at
13 http://www.horus.com/~hias/atari/ - either as source code (being part of the
14 'AtariSIO driver and utilities for Linux') or as Win32 binary program (being
15 part of the 'Atari Tools for Win32').
